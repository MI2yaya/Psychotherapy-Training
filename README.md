To run website locally, run GUI 2/app.py. May require training of models to run Chatbots.



For training, requires .env file in following format:

GOOGLE_APPLICATION_CREDENTIALS=...json (key for gemini training)

BOTO_PATH=... (key generated from gsutil)

CLAUDE_KEY=... (Claude key)

GOOGLE_KEY=... (Google API key)

GOOGLE_PROJECT=... (Google project key)

LLAMA_KEY=... (Meta's Llama key)

NEBIUS_KEY=... (Nebius model key)

OPENAPI_KEY=...
